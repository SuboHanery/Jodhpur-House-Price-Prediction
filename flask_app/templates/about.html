About The Project

Bringing transparency and data-driven decision-making to the housing market in Jodhpur.

The real estate market is often clouded by subjective pricing and opaque broker estimations. By leveraging Machine Learning, we aim to provide buyers, sellers, and realtors with objective, instant, and highly accurate property valuations tailored specifically to Jodhpur's unique localities.

The Technology Stack

Frontend UI/UX

Built with Modern HTML5 and Vanilla JS. The interface utilizes a premium Glassmorphism aesthetic with custom CSS3 gradients, frosted glass panels, and smooth micro-animations to deliver a SaaS-like experience.

Backend API

Powered by Python and the Flask framework. The backend provides a lightweight, lightning-fast RESTful API (`/api/predict`) that seamlessly connects the frontend interface to the machine learning engine.

AI Engine & Data

The core intelligence is driven by Scikit-Learn and Pandas. Our predictive model is a heavily-trained Random Forest Regressor that analyzes 28 distinct features (BHK, Age, Amenities) to estimate property values.

Deployment Base

Configured to be production-ready using robust WSGI servers like Waitress or Gunicorn, ensuring the application remains highly available and performant even under concurrent load.
